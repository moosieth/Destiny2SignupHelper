### Added

- A Python project for a Discord bot that can create LFG postings for raids and dungeons
  - Has commands:
    - `/ping` - Gets a simple response
    - `/lfg-raid` - Create an LFG posting for a Raid activity
    - `/lfg-dungeon` - Create an LFG posting for a Dungeon activity
    - `/help` - Get a help message
